# sistema-preaprobacion-bancario
Sistema de pre-aprobación de prestamos bancarios. Implementando validaciones lógicas y control de excepciones. 

## Tecnologias:
- Java 21 (Uso de records y Switch Expresions).
- Maven

## Funcionalidades
- Calcular cuotas: Aplica intereses segun el tipo de prestamo que se desea sacar.
